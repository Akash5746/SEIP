SEIP - Smart Expense Intelligence Platform
==========================================

Enterprise-grade microservices platform for automated expense management,
fraud detection, and compliance. Built with Java 21, Spring Boot 3.3, React 18,
Kafka, PostgreSQL, MinIO, and full DevOps pipeline.

QUICK START
-----------
1. Prerequisites: Docker Desktop, Java 21, Node.js 20, Python 3.11

2. Start infrastructure:
   docker-compose -f docker-compose.infra.yml up -d

3. Build all Java services:
   ./mvnw clean package -DskipTests

4. Start all services:
   docker-compose up -d

5. Access the application:
   Frontend:   http://localhost:3000
   API Gateway: http://localhost:8080
   Kafka UI:   http://localhost:8090
   MinIO:      http://localhost:9001
   Grafana:    http://localhost:3001
   Kibana:     http://localhost:5601

ARCHITECTURE NOTES
------------------
- API Gateway handles JWT validation for all downstream services
- Services trust X-Auth-User-Id and X-Auth-User-Role headers from Gateway
- All services expose /actuator/prometheus for Prometheus scraping
- JSON structured logging via Logback for ELK ingestion
- MinIO provides S3-compatible receipt storage locally
